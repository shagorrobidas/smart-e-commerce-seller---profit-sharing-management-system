# Smart E-Commerce Seller & Profit Sharing Management System

A web-based business management platform designed for small to medium e-commerce sellers.

## Features
- **Centralized Dashboard**: Manage sales, expenses, and inventory from one place.
- **Profit Calculation**: Automatic computation of Net Profit based on COGS and expenses.
- **Investor Transparency**: Dedicated portal for investors to view real-time performance and payouts.
- **Role-Based Access**: Secure access for Admins, Staff, and Investors.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Storage**: LocalStorage (for demonstration purposes)

## Setup
1. Open `index.html` in your browser.
2. Navigate to "Get Access" to create an account.
